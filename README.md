## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/Python-Fundamentals.git
```

### 2. Open the Project

```bash
cd Python-Fundamentals
```

### 3. Run Any Python File

```bash
python filename.py
```

or

```bash
python3 filename.py
```
